# MNIST Digit Recognizer

A simple handwritten digit recognition project built with TensorFlow and the MNIST dataset.

## Project Overview

This project trains a neural network to recognize handwritten digits (0-9) using the MNIST dataset.

The model was trained using TensorFlow/Keras and achieved high accuracy on the test dataset.

## Features

* Load and preprocess MNIST data
* Train a neural network model
* Evaluate model performance
* Save and load trained models
* Predict custom handwritten digit images

## Results

* Training Accuracy: 98.58%
* Test Accuracy: 97.35%
